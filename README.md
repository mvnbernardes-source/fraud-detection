# Descrição

Projeto de Machine Learning para detecção de fraudes em transações de crédito, com foco em identificar o maior número possível de fraudes em um cenário de dados desbalanceados.

# Base de Dados

Devido ao tamanho do arquivo, o dataset não está incluído no repositório.
Link: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

# Metodologia

Análise exploratória, tratamento dos dados, divisão em treino e teste, treinamento e avaliação de modelos de classificação.

# Modelos e Resultados

Foram avaliados Regressão Logística e Random Forest. A Regressão Logística apresentou maior ROC AUC, enquanto o Random Forest obteve maior recall para a classe de fraude.

# Conclusão

Como o objetivo é reduzir fraudes não detectadas, o Random Forest foi escolhido como modelo final.
